### Start.bat for Minecraft server!
### Parameters that need to be entered into Start.bat
###                 ⬇⬇⬇⬇

java -Xmx1024M -Xms1024M -jar server.jar

### Description of each parameter
###                 ⬇⬇⬇⬇

java - java notation

-Xmx1024M -Xms1024M - min/max allocated RAM

-jar - Don't touch this

server.jar - Server core name
